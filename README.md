# Oireachtas_RAG
RAG model to query Dáil debates. This is a personal project. I came across the Oireachtas database and API: https://data.oireachtas.ie/ and saw the availibilty of information.
I am now trying to build a RAG model whereby, one can query the model to get TDs positions on issues, maybe find contradictions in things TDs have said. It could be interesting to 
add voting information later on.
